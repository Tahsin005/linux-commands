grep,  egrep,  fgrep, rgrep - print lines that match patterns

    grep [OPTION...] PATTERNS [FILE...]
    grep [OPTION...] -e PATTERNS ... [FILE...]
    grep [OPTION...] -f PATTERN_FILE ... [FILE...]

    grep "happiness" life.txt   -> case sensitive
    grep -i "happiness" life.txt  -> case insensitive
    grep -c "happiness" lifet.txt  -> counts the number of occurences
    grep -in "happiness" life.txt   -> show the line no if matches

    grep -iw "Hi" hello.txt  -> searches as a word
    grep -iB2 -A3 "tom" cats.txt  -> shows 2 line before and 3 lines after the match

        grep -iC2 "tom" cat.txt  -> considers before and after the match

    grep -r "tahsin"   -> recursively searches on each files in the directory
    grep -r "tahsin" Documents/
